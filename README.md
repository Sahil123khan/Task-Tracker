# ✅ Task Tracker

A simple and interactive task management application built with React and Tailwind CSS. You can add, delete, complete, and filter tasks. Also uses `localStorage` to save your data.

## 🚀 Live Demo  
👉 [Click here to view the live project](https://courageous-donut-92f1b7.netlify.app/)

## 🛠️ Features

- Add new tasks  
- Mark tasks as complete  
- Delete tasks  
- Filter tasks (All, Active, Completed)  
- Data stored in localStorage

## 🧑‍💻 Tech Stack

- React.js  
- Tailwind CSS  
- JavaScript  
- HTML & CSS
